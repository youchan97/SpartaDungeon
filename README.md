# SpartaDungeon
스파르타 던전 탐험기

# 게임 개요


다양한 기믹과 아이템을 배치시켜 꼭대기까지 도달하라!!


# 코드


1. Player

+ 플레이어는 InputSystems를 통해 이동, 회전, 점프 등 움직임을 사용한다.
+ 각각의 상태를 FSM으로 관리하여 확장성을 염두해두었다.
+ 상태별로 애니메이션 전이를 사용하였다.

2. Item

+ 아이템은 Item을 상속받아 아이템 타입에 따라 효과를 적용시킨다.
+ ItemType은 현재 Consume = 사용, Buff = 버프 로 나뉘어있다.

3. Buff

+ 버프는 버프 아이템을 플레이어가 획득 시 BuffManager에서 플레이어에게 버프를 주는 형식
+ 아이템과 버프의 책임을 분리하여 SRP의 원칙을 중요시 했다.(아이템은 획득 or 사용만, 버프는 플레이어에게 버프를 걸어줌)

4. Obstacle

+ 장애물의 경우 확장을 위해 상속받을 수 있게 짜놨다.
+ 장애물의 데미지 타입이 존재 단타/도트데미지로 짜놨다.

5. SingleTonManager

+ GameManager, UiManager, SoundManager 등 게임 내내 사용할 것들을 싱글톤화해서 사용하였다.
+ 각각의 스크립트에서 싱글톤을 가져와야 할 경우 되도록 캐싱하여 사용하도록 하였다.

6. Loading

+ LoadAsync를 사용해 로딩씬에서의 로딩을 구현하였다.
+ 현재 로딩이 한프레임만에 되기때문에 로딩씬을 보여주기 위해 로딩따로 로딩바따로 연출 구현을 하였다.
